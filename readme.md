### `mask-attr-feather'
----

Replaces `data-feather` nodes with SVG icons

```mask
button {
    i data-feather=circle;
}
```


#### Install

```bash
# peerDependency
> npm i feather-icons
> npm i mask-attr-feather
```
```mask
import '/node_modules/data-feather/lib/attr.js'
```
